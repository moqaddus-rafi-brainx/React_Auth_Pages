#React Authentication Practice Project

This is a React-based practice project focused on implementing core authentication-related pages. It's built to help understand form handling, validation, and conditional rendering using React.

##Features

- Signup Page
- Login Page
- Forgot Password Page
- Reset Password Page
- Verify Password Page
- Form validation using `useReducer`
- Clean, modular code structure
- Built entirely with React (functional components + hooks)
